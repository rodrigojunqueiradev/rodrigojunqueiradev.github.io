Controle de Orçamento Pessoal

22 de Abril de 2024 por Rodrigo Junqueira

Introdução

Todos os nomes envolvidos nesta publicação são aleatórios.

Em um trabalho de freelancer, a cliente fez a seguinte solicitação:

"Criação de Modelo de Excel para Acompanhamento de Finanças Pessoais Este projeto envolve a criação de um modelo de Excel que funcione como uma planilha para o acompanhamento de receitas e despesas, especificamente para o gerenciamento de finanças pessoais. O principal objetivo é desenvolver uma ferramenta fácil de usar e personalizável que permita aos usuários registrar suas receitas, despesas e orçamento de maneira eficaz. O modelo de Excel pode incluir recursos como planilhas separadas para receitas e despesas, categorização de gastos, gráficos e visualizações, funcionalidade para estabelecer metas financeiras e outros recursos relevantes. Confiamos em sua experiência para incluir o que achar necessário!"

  • Um campo de texto livre para o nome da receita;

Para a guia de despesas, desenvolvi um controle mensal, onde o usuário tem:

  • Um campo de texto livre para o nome da despesa;
  • Um campo para o tipo da despesa, que é composto por uma lista padronizada, conforme a imagem abaixo:
  • Cálculo somatório automático organizado por:
    • Bimestral
    • Trimestral
    • Semestral
    • Anual

Guia de Orçamento

A guia de orçamento é um dashboard, em layout pronto para impressão ou para salvar em PDF, de todos os dados informados nas guias de Receitas e Despesas.

O dashboard possue filtros de período e por quais tipos de receitas e despesas.

Guia Modalidades

A planilha ainda disponibiliza ao usuário uma guia chamada "Modalidades", onde ele tem a possibilidade de incluir/alterar/excluir os tipos padronizados de receitas e despesas.
