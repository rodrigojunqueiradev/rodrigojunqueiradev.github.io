Public Data Analysis of Elderly Care Units

January 31, 2025 by Rodrigo Junqueira

Introduction

A private client requested an analysis of data from elderly care units in the Censo Suas, conducted by the government and publicly released on gov.br.

The client is a writer and needs descriptive and statistical analyses of the data.

The client provided me with 18 questions that needed to be answered in his writing, requesting various types of tables and charts for each question. Based on the results, he would develop his text and include the charts he found relevant.

The analyses needed to be temporal, comparing the years from 2019 to 2023.

The publicly available data contains a lot of noise and is difficult to interpret. Therefore, before conducting a detailed analysis, the client needed the data source to be cleaned.

All data cleaning and transformation were performed using the Python programming language.

All analysis was conducted in Excel, the client’s platform of choice.

Project Stages

The following steps were taken for the project development:

  1. Analysis of public data.
  2. Video call with the client to agree on the project stages.
  3. Data collection, cleaning, and transformation using Python.
  4. Statistical analysis.
  5. Final delivery.
