Análise de dados públicos de unidades de acolhimento de pessoas idosas

31 de Janeiro de 2025 por Rodrigo Junqueira

Introdução

Um cliente particular solicitou a análise dos dados das unidades de acolhimento de pessoas idosas do Censo Suas, realizados pelo governo e divulgado publicamente nos dados do gov.br.

Tal cliente é um redator e precisa de análises descritivas e estatisticas dos dados.

O cliente me passou 18 questões que ele precisa responder em sua escrita, pedindo que fosse desenvolvido para cada uma destas questões diversos tipos de tabelas e gráficos, assim com base nos resultados ele iria desenvolver sua redação e incluir os gráficos que ele escolher nelas.

As análises precisam ser temporais, comparando os anos de 2019 até o ano de 2023.

Os dados públicos disponibilizados contém muito lixo e são de difícil interpretação, desta forma, antes de uma análise detalhada o cliente precisava que a fonte de dados fosse limpa.

Toda a limpeza e transformação dos dados foi realizada utilizando a linguagem de programação Python.

Toda a análise foi realizada no Excel, plataforma de escolha do cliente.

Etapas do trabalho

Para o desenvolvimento do projeto, foram tomadas as seguintes etapas:

  1. Análise dos dados públicos.
  2. Chamada de vídeo com o cliente para acordarmos as etapas do projeto.
  3. Coleta, limpeza e transformação da fonte de dados usando Python.
  4. Análise estatística.
  5. Entrega final.

Desenvolvimento

A limpeza e transformação dos dados foram realizadas utilizando O Jupyter Notebook, com a linguagem de programação Python e a biblioteca Pandas.

Durante o processo de limpeza e transformação, ajustei todos os tipos de insconsistências na fonte de dados e gerei uma nova limpa e unificada entre os 5 anos de análise.

No Excel, criei 18 abas que seriam as perguntas solicitadas pelo cliente, com diversos tipos de gráficos e tabelas.

Entrega

O resultado do projeto foi o arquivo em Excel, com explicações e detalhamentos de todas as variáveis que o cliente precisava responder.
